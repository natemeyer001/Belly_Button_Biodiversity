Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

The filter on the left side of the page allows you to choose a new test subject besides 940. When a new id is selected, the page will automatically reload with the 3 plots updated with that subject's data. The first graph is a gauge chart that displays how often the test subject washed their belly button. To the right of that is a horizontal bar chart displaying the most frequent bacteria cultures found in that subject. Note: you can hover over the bacteria to get the count as well as the name. Lastly, on the bottom is a buble chart of the bacteria cultures. This chart also displays the count and name when you hover over a circle. The size of the circles signifies the count of the bacteria culture, where the larger the circle, the higher the bacteria count.
